1er Trimestre – Diseño Web: GitHub y Hosting

1. ¿Qué es GitHub?

GitHub es una plataforma web donde los desarrolladores pueden guardar, compartir y colaborar en proyectos de código.
Está basada en Git, un sistema de control de versiones que permite llevar un historial de los cambios que haces en tus archivos.

2. ¿Cómo crear una cuenta en GitHub?

- Ve a github.com
- Llena los datos: usuario, correo y contraseña
- Verifica tu correo
- ¡Cuenta lista!

3. ¿Qué es Hosting?

Es un servicio que permite subir tu página a Internet para que otros la vean. GitHub Pages es uno gratuito.

4. Ejemplos de Hosting

  1. GitHub Pages: Hosting gratuito para sitios estáticos (HTML, CSS, JS). Ideal para proyectos personales. Se conecta directamente con tus repositorios.
  2. Netlify: Hosting gratuito con funciones avanzadas para sitios JAMstack. Despliegue automático desde GitHub, certificados SSL incluidos.
  3. Vercel: Ideal para proyectos con frameworks modernos como Next.js. Rápido, gratuito y escalable.
  4. Bluehost: Hosting de pago compatible con WordPress y PHP. Incluye dominio gratis por un año y soporte 24/7.
  5. HostGator: Hosting económico con panel fácil de usar. Bueno para blogs y páginas pequeñas.
  6. Google Cloud / AWS / Azure: Plataformas en la nube para grandes proyectos. Ofrecen planes gratuitos para comenzar.
